### <a name="Emphahis"></a>Выделение элементов
Текст выделен **полужирным**. И этот текст тоже будет __полужирным__.

Текст выделен *курсивом*. И этот текст тоже будет _курсивом_.

***

***А что тут будет???***

Этот `cmd` внутри абзаца

    < cmd_2.0 >
    код - не кот
    кот - не смог

Ещё одна вариация:

```
1.2.3.4.5
я@ты@я@ты@я@
ладно
```
